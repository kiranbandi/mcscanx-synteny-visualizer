SynVisio : MCScanX Synteny Visualizer Tool

SynVisio is an interactive multiscale visualization tool that lets you explore the results of McScanX a popular synteny and collinearity detection toolkit. SynVisio requires two files to run:

  • The simplified gff file that was used as an input for a McScanX query.
  • The collinearity file generated as an output by McScanX for the same input query.

SynVisio offers two basic forms of visualizations a Linear bar plot and a Dot plot. General information regarding the paramters that were set for McScanX and the percetage share of collinear genes, are read from the collinearity file and displayed above with the plots. Users can choose the source and the target chromosomes and the type of the plot using the options panel.Users also have a theming option that they can choose which toggles the background of the plot making it easier to visualize outlier points and links.

Linear Bar Plot

This plot visualises chromosomes as horizontal bars and connects genetically collinear regions between source and target regions with links. There are two levels of view for this plot a whole genome level view and a chromosome level view. The genome view uses colors to distniguish different chromosomes and can be used to see which source and target chromosomes share collinear regions.The width of the links represents the length of the sequences that are collinear.Hovering the mouse over a source chromosome displays all the links for which that chromosome is a source. Clicking on a source chromosome marker displays only the links emerging from that chromosome.Double clicking on any marker resets this effect. To investigate a region further users can click on a source chromosome and then click on a target chromosome to invoke the chromosome view for that particular source and target chromosome. The chromosome view uses colour to distniguish between a direct alignment and a reversed / flipped alignment. Users can use the mouse to zoom and pan the plot to investigate links.Hovering over a link also displays additional information regarding the link like the E value , match score and the count.Double clicking on a link highlights that particular link and hides all the other links. The reset icon at the top right corner can be used to reset the plot to its original state.

Dot Plot

A dot plot is simply the two dimensional representation of the similarity matrix between two sequences. The source chromosomes and target chromosomes are on the X and Y axes respectively and a link between two regions is represented by a dot.If the link is longer its represented by a line the length of which depends on the length of the match. This plot like the chromosome view in the linear Bar plot can be zoomed and panned for closer investigation.Hovering over a dot or a line shows the users additional information regarding that link.
